Introduction to Python Programming:

Final Project

This final project replaces the requirement for a final exam of this course.

 Updated 2/21/2023

  1. Choose a programming task or problem that can be completed in a few hours of work. Examples include: analyzing some real-world data of your interest, image processing, text analysis, solving puzzles, creating a simple text-based computer game, exploring advanced features of Python modules, etc.

  2. Develop a Python code that achieves the task or solves the problem. [NOTE: Don’t plagiarize the coding examples available online!!] Plagiarism checks will be performed on all submissions.

 

  1. Prepare a report of your work. Include your Python code and code output, written explanations of your results, and document your code (such as including brief comments within the code that help the reader understand your code better). Make sure to show the grader that you understand what you are doing and why you are doing it. In the explanation, avoid just walking through your code and instead try to provide an explanation for why you are doing what you are doing, and why it leads to the results it does.
  2. Use Jupyter Notebook, Google Collab, or similar environment. You can only submit one file so you must combine your work into one file. Save the report in *PDF and submit it on Brightspace in the Content folder via Turnitin by the course end date. Screenshots are NOT ALLOWED. You should be able to export your code as a PDF. Your file should be scannable by Turnitin.
  3. Click on the Final Project Upload to start the upload of your project. You can refer to the following article for PDF conversion in Jupyter Notebook. URL: https://towardsdatascience.com/jupyter-notebook-to-pdf-in-a-few-lines-3c48d68a7a63#:~:text=In%20your%20notebook%2C%20click%20the,HTML%20to%20transform%20the%20notebook. OR refer to the first option at https://mljar.com/blog/jupyter-notebook-pdf/#:~:text=The%20Jupyter%20Notebook%20has%20an,not%20visible%20in%20the%20screenshot).

* If you have difficulty in saving your Jupyter notebook as a PDF file, try printing it into a PDF file using your browser’s “Print” feature (or save the notebook as HTML and then convert the HTML file into PDF).
